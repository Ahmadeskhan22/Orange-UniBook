

UniBook is a comprehensive mobile application built with Flutter designed to revolutionize how university students in Jordan find and purchase textbooks. By centralizing academic resources, it provides a seamless, secure, and efficient marketplace tailored to student needs.

üöÄ Key Features

Secure Authentication: Robust user registration and login flow powered by Firebase Authentication.

Dynamic Book Catalog: Real-time browsing and searching across thousands of academic titles via the Google Books API.

University Filtering: Personalized experience allowing users to filter books based on specific Jordanian universities.

Smart Cart Management: Efficient state management using Provider to handle adding, removing, and updating cart items.

Auto-Location Detection: Integrated Location and Geocoding services to automatically fetch the user's delivery address for a faster checkout.

Complete Checkout Flow: A structured ordering process with backend API integration and order status tracking.

Modern UI/UX: Clean, responsive design with rounded edges, subtle shadows, and intuitive navigation.
